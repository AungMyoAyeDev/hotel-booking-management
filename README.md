#  Hotel Booking Management System

Welcome to the Hotel Booking Management System. This repository serves as the central umbrella project, orchestrating the frontend, admin dashboard, and backend services through Git submodules.

##  Project Structure

This project adopts a modular architecture, separated into three distinct repositories:

* **[hotel-booking](https://github.com/AungMyoAye101/hotel-booking.git)**: The public-facing client application where users can browse available rooms and make reservations. Built for responsive UI performance using React and Next.js.
* **[booking-dashboard](https://github.com/AungMyoAye101/booking-dashboard.git)**: The administrative control panel used by hotel staff to manage bookings, view analytics, and update room availability.
* **[hotel-booking-backend](https://github.com/AungMyoAye101/hotel-booking-backend.git)**: The core backend API and database layer that powers both the frontend application and the admin dashboard, engineered with Go for scalability and robust performance.
